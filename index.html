LazyOwn Framework [;,;]

About

LazyOwn Framework is a powerful Framework written in Python. It, Fuzzing, Webshellz, BotNet, Rat, Keylogger, Nmap, Pwn, and captures and analyzes network packets using the Scapy library and provides an interactive terminal interface with curses.

Installation

To get started with LazyOwn Framework, clone the repository and install the dependencies:

git clone https://github.com/grisuno/LazyOwn.git
cd LazyOwn
pip install -r requirements.txt

Usage

Run the Framework with the following command:

python app.py
